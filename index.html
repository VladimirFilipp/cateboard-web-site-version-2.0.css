Сылва

Сплав по данной реке идеально подходит семьям с детьми, а так же людям с нулевым туристическим опытом. Спокойная теплая река, с множеством стоянок, отсутствием экстримальных моментов и достаточным для неподготовленного туриста количеством достопримечательностей.

Список достопримечательностей: гора Лобач (1 день сплава), мраморные каньоны + парк под открытым небом + навесной мост + посещение деревенского магазина – (2 день, все эти места посещаются подряд в предобеденное время второго дня), гора Ермак (3день).

Продолжительность сплава 3 дня

Усьва

Сплав для более подготовленных туристов, речушка побыстрее, встречаются расчески, в летний сезон возможно обмеление реки ( придется тащить катамаран на руках или идти по скользким булыжникам рядом с катамараном, чтобы он не стоял посреди реки

По достопримечательностям всем известные Усьвинские столбы (1 день), камень Панорамный и камень Омутной (2 день), + Живописные таежные виды и достаточно быстрое течение.

Продолжительность сплава 3 дня ( для особо занятых людей можно организовать 2-хдневный сплав).
